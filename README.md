# Uber Control Pro

PWA premium para conductores de Uber que quieren medir ingresos, costos operativos y ganancia real.

## Abrir localmente

Puedes abrir `index.html` directamente, aunque para probar instalación PWA y service worker conviene servir la carpeta con un servidor local.

## Incluye

- Dashboard con KPIs de ingresos, gastos, kilómetros, viajes, ROI, ganancia por hora y ganancia por KM.
- Cálculo separado de ganancia operativa y ganancia real.
- Reserva de mantenimiento y depreciación por kilómetro configurables.
- Registro diario con resultados en tiempo real.
- Reportes semanal, mensual y anual.
- Módulo Kia K5 2015 con salud del vehículo y mantenimiento próximo.
- Exportación CSV, JSON y archivo compatible con Excel.
- Importación CSV y JSON.
- `manifest.json` y `sw.js` para instalación como PWA.

## Publicación en GitHub Pages

1. Crear un repositorio llamado `uber-control-pro`.
2. Subir estos archivos a la rama principal.
3. Activar GitHub Pages desde la rama principal y la carpeta raíz.

La URL quedará con el formato `https://USUARIO.github.io/uber-control-pro/`.
